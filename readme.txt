== Description ==

Enables you to hide the sidebar Widgets you don't need.
This plugin cleans your Widgets creen :)

== Installation ==

1. Upload the entire `unregister-widgets` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to /wp-admin/themes.php?page=unregister_widget to choose which Widgets you want to hide.

== Frequently Asked Questions ==
